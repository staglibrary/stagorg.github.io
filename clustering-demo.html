Local Graph Clustering with Conflict Data


World Map

Date range: 1900 - 1950

Cluster size factor: 0.5

Instructions

Use the sliders to select a time period and cluster size. Then, click a country on the map to see two groups of countries which often had conflicts with each other between the chosen dates.

Date range: the country groups are based on data of conflicts between the given dates.

Cluster size factor: the cluster size factor controls the parameters of the clustering algorithm. A smaller factor will find smaller groups which are more closely related to the selected country.

Key

- the selected country

- the countries in the same group as the selected country.

- the countries in the other group.

Dataset

The dataset used in this application is the Correlates of War Dyadic Militarized Interstate Disputes Dataset which can be downloaded from here. This dataset contains a record of every militarized dispute between 1816 and 2014, and includes information about the countries involved and the severity of the dispute.

Algorithm

The algorithm for finding groups of countries is based on Local Graph Clustering. We construct a graph based on the militarized disputes between the chosen dates. Every country is represented by a vertex in the graph, and there is an edge between each pair of countries weighted according to the severity of their military disputes. Then, by performing random walks on the graph, it is possible to recover groups of countries which tend to enter conflicts with each other. Roughly speaking, the cluster size factor is used to control the length of the random walks, and shorter walks result in smaller clusters which are more closely related to the starting country.

Code

The code for computing the clusters is available on GitHub. The algorithm uses the approximate_pagerank method of the STAG Library in order to compute the stationary distribution of the personalised Pagerank on the constructed graph.

Reference

Peter Macgregor, He Sun: Local Algorithms for finding densely connected clusters. ICML 2021: 7268-7278
