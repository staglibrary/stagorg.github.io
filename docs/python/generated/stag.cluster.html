stag.cluster.local_cluster(g: LocalGraph, seed_vertex: int, target_volume) → List[int]#

Default local clustering algorithm.

Given a graph and starting vertex, return a cluster which is close to the starting vertex. The target_volume parameter controls the size of the returned cluster.

You should set target_volume to be your best guess for the volume of the cluster you would like to find. For example, the following code will find one of the ‘clusters’ in a barbell graph.

import stag.graph
import stag.cluster
graph = stag.graph.barbell_graph(5)
cluster = stag.cluster.local_cluster(graph, 1, 21)

This method calls through to the local_cluster_acl() method.

Parameters:

  • g – a stag.graph.LocalGraph object

  • seed_vertex – the starting vertex in the graph

  • target_volume – the approximate volume of the target cluster

Returns:

a list of vertices in the same cluster as the seed vertex

stag.cluster.local_cluster_acl(g: LocalGraph, seed_vertex: int, locality: float, error=0.001) → List[int]#

The ACL local clustering algorithm.

Given a graph and starting vertex, returns a cluster close to the starting vertex.

The locality and error parameters correspond to the \(\alpha\) and \(\epsilon\) parameters of the ACL algorithm.

Parameters:

  • g – a graph object implementing the LocalGraph interface

  • seed_vertex – the starting vertex in the graph

  • locality – a value in \((0, 1]\) indicating how ‘local’ the cluster should be. A value of \(1\) will return only the seed vertex and a value of \(0\) will explore the whole graph.

  • error – (optional) the acceptable error in the calculation of the approximate pagerank. A smaller error will result in longer running time and higher quality cluster.

Returns:

a list containing the indices of vertices considered to be in the same cluster as the seed_vertex.

Reference:

[ACL] Andersen, Reid, Fan Chung, and Kevin Lang. “Local graph partitioning using pagerank vectors.” FOCS’06.

stag.cluster.approximate_pagerank(g: LocalGraph, seed_vector: csc_matrix, alpha: float, epsilon: float) → Tuple[csc_matrix, csc_matrix]#

Compute the approximate pagerank vector.

The approximate pagerank vector \(\mathrm{apr}_G(s, \alpha, \epsilon)\) is defined in [ACL] and this method implements their proposed algorithm for computing it.

This method forms an important part of the local_cluster_acl() algorithm.

Note that the dimension of the returned vectors may not match the true number of vertices in the graph provided since the approximate pagerank is computed locally.

Parameters:

  • g – a stag.graph.LocalGraph

  • seed_vector – a sparse column matrix defining the starting distribution on the graph. Although this is a matrix type, it should have exactly one column.

  • alpha – a value in \((0, 1]\) controlling the teleportation parameter of the personalised Pagerank.

  • epsilon – a value in \((0, 1]\) controlling the approximation guarantee.

Returns:

  • \(p\) : the approximate pagerank vector

  • \(r\) : the residual vector

Raises:

ArgumentError – if the provided seed_vector is not a column vector.

Reference:

[ACL] Andersen, Reid, Fan Chung, and Kevin Lang. “Local graph partitioning using pagerank vectors.” FOCS’06.

stag.cluster.sweep_set_conductance(g: LocalGraph, v: csc_matrix) → List[int]#

Find the sweep set of the given vector with the minimum conductance.

First, sort the vertices such that \(v(1) \leq v(2) \leq \ldots \leq v(n)\) . Then, let

\[S_i = \{j : j <= i\}\]

and return the set of original vertex indices that correspond to

\[\mathrm{argmin}_i \quad \phi(S_i),\]

where

\[\phi(S_i) = \frac{w(S_i, \overline{S_i})}{\mathrm{vol}(S_i)}\]

is the conductance of \(S_i\) .

This method is expected to be run on vectors whose support is much less than the total size of the graph. If the total volume of the support of v is larger than half of the volume of the total graph, then this method is likely to return the total support of v.

Parameters:

  • g – the stag.graph.LocalGraph on which to operate

  • v – a sparse column vector

Returns:

a list of the indices of v which give the minimum conductance
