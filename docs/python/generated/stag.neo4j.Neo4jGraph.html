stag.neo4j.Neo4jGraph#

class stag.neo4j.Neo4jGraph(uri, username, password)#

Bases: LocalGraph

Represent a neo4j database as a STAG LocalGraph object.

This provides ‘local’ access to the graph only. Given a node, we can query its neighbors.

For now, the graph does not support weighted edges, and it does not distinguish between edge types or directions.

degree(v)#

Given a vertex v, return its weighted degree.

Given a graph \(G = (V, E, w)\), the degree ov vertex \(v\) is defined by

\[\mathrm{deg}(v) = \sum_{u \in V} w(v, u).\]

degrees(vertices: List[int]) → List[float]#

Given a list of vertices, return a list of their weighted degrees.

When developing implementations of the LocalGraph class, providing an efficient method of returning a list of degrees will improve the performance of local clustering algorithms.

neighbors(v) → List[Edge]#

Given a vertex v, return a list of edges representing the neighbors of v.

The returned edge objects will all have the ordering (v1, v2) such that edge.v1 = v and edge.v2 is a neighbor of v.

neighbors_unweighted(v) → List[int]#

Given a vertex v, return a list of neighbors of v.

The weights of edges to the neighbors are not returned by this method.
