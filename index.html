STAG Library

Spectral Toolkit of Algorithms for Graphs

This webpage reports our progress developing STAG, an open-source C++ library for efficient spectral algorithms.

About Us

STAG is a C++ library providing efficient spectral algorithms for the analysis of massive graphs. The development team of STAG is based in the School of Informatics, University of Edinburgh, United Kingdom.

Code

You can find STAG's source code from our Github page .

FAQ

Frequently Asked Questions

  • Which programming language is used to develop STAG?

    STAG is mainly written in C++. However, it contains a python wrapper STAGPy around the underlying C++ library which provides convenient methods for learning on graphs in python.
